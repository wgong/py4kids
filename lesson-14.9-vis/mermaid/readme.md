
# [Mermaid](https://github.com/mermaid-js/mermaid)

- [Live Editor](https://mermaid.live/)
- [Doc](https://mermaid.js.org/intro/)

```mermaid
graph TD;
    A[Start] --> B[Decision]
    B -- Yes --> C[Option 1]
    B -- No --> D[Option 2]
    C --> E[Eval]
    D --> E
    E[Eval] --> F[Stop]
```
generates this diagram
